# Build

Windows - `yarn package-win`
MacStore - `./build-mas.sh`
Mac - `./build-mac.sh`

Build all - `./build.sh`


For Windows:
WIN_CSC_LINK: link to the win certificate link
WIN_CSC_KEY_PASSWORD: the password used to encrypt the win certificate

For Mac:
Certificates in keychain
